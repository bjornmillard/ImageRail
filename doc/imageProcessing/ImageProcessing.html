Image Processing

Preprocessing

ImageRail, in its default form, does not perform image pre-processing such as image flat-field calibration, contrast enhancements, or various convolution filtering. These features can be implemented by others as segmentation plug-ins if desired.
For our purposes when we have needed preprocessing, we use the NIH's free, open-source ImageJ program.
